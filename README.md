# Top-Gear-Lite
Uma página web no formato de tier list elencando modelos de carros esportivos lançados na última década (2010-2020)


Este projeto trata-se de uma página web, simulando uma versão Lite do Site de noticias automotivo Top Gear, onde listo grandes lançamentos de carros esportivos com um mecanismo de busca por palavras chave, interatividade e responsividade, tratamento de erros, imagens, descrição e link que redirecionam o leitor para a página oficial do Top Gear onde há a matéria completa de descrição de cada modelo.

Observações importantes: Este projeto trata-se de um estudo de caso promovido pela Alura em um programa em parceria com o Google que visa trazer fundamentos e exemplos da utilização do Gemini como ferramenta de assistente de código e API. Este projeto foi concebido com o auxílio dos apresentadores da Imersão Dev Google, dos materiais disponibilizados na imersão e do próprio Gemini Code Assist para correções e implementações de funcionalidades nos códigos.
